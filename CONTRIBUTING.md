# Contributing

Contributions are welcome via GitHub pull requests. By contributing, you agree
that your contributions will be licensed under the same MIT-0 license as the rest
of this project.

## Reporting bugs / feature requests
Please open a GitHub issue with a clear description and reproduction steps.

## Pull requests
1. Fork the repository
2. Create a feature branch from `main`
3. Make your changes; include tests where applicable
4. Open a PR with a clear title and description

## Security
If you discover a potential security issue, please follow SECURITY.md — do NOT
open a public GitHub issue.
